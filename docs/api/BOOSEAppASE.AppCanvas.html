A class creating a canvas interface for BOOSE, implementing BOOSE.ICanvas. Provides a number of methods for drawing/editing including Circle(int, bool), Rect(int, int, bool), MoveTo(int, int) and DrawTo(int, int).

Circle(int, bool)

Draws a circle to the canvas around the current pen position. The size of the circle is determined by the radius. The circle can either be an outline or filled.

Rect(int, int, bool)

Draws a rectangle to the canvas, starting from the current pen position. The size is determined by the given width and height values. The rectangle can either be an outline or filled.

public void Rect(int width, int height, bool filled)

The width value of the rectangle, used to determine the rectangle's size.

Exceptions

Set(int, int)

Creates and sets the size values for a new canvas. Any previous graphics objects will be dropped first. The width and height will be set, along with providing a clear white canvas. The pen position will automatically be set to the top left position (0,0).

Parameters
