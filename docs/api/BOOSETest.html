Unit tests for AppFor. Makes sure that both increasing and decreasing loops with positive and negative steps, execute as expected.

Unit tests for AppIf. Tests that the correct potion of the if statement is executed depending on whether the condition is true or false.

Unit tests for AppInt. Tests to make sure that integers are stored and declared correctly, with duplicates throwing an exception.

Unit tests for AppParser. These check that the parser is correctly handling commands, including if statements and loops.

Unit tests for AppPeek. Makes sure that peek correctly reads the selected array value, assigning it to the variable.

Unit tests for AppPoke. Checks that poke correctly writes values into arrays whether they are integer or real variable types.

AppRectTests

Unit tests for AppRect. Tests that rectangle commands can be executed correctly, whether the parameters are variables or numbers.

Unit tests for AppValueFix. These tests check that integer values can be resolved from both variables and expressions.

Unit tests for AppWhile. Makes sure that while loops run as expected, with the stated number of iterations.

Unit tests for AppWriteText. Makes sure that it can write correctly using expression results and concatenation.
